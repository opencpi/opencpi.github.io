Prerequisites

Before installing following the commands below, you will need EPEL enabled:
  • yum install epel-release

Yum repositories

opencpi-v1.5.0.repo - Copy to /etc/yum.repos.d/ as root user and then yum install opencpi or yum install 'opencpi-*'.
opencpi-v1.4.0.repo - Copy to /etc/yum.repos.d/ as root user and then yum install opencpi or yum install 'opencpi-*'.
opencpi-v1.3.1.repo - Copy to /etc/yum.repos.d/ as root user and then yum install opencpi or yum install opencpi-devel 'ocpi-prereq-*'.

Example

This example works for 1.4 and later, always pulling the current version.
  • cd /etc/yum.repos.d/ && curl -O http://opencpi.github.io/repo/opencpi.repo && yum install -y epel-release && yum install -y 'opencpi-*'
